Musi is a programming language that aims to be safe and easy to use.

## Building from Source

- [Git](https://git-scm.com) (optional)
- [XMake](https://xmake.io/#/)
- [Clang 13+](https://clang.llvm.org)

```bash
xmake project -k compile_commands
xmake f -m release
xmake
```
