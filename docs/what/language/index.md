Musi is easier to learn when each new idea arrives alone.

## What

This book teaches current Musi from first setup through methods, effects, runtime, stdlib, and tooling.

## Why

Many guides overload beginners by mixing syntax, types, effects, and packages too early.

## How

Read in order when Musi is new. Skip only when the earlier model already feels natural.
