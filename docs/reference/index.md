# The Musi Language Reference

This guide explains how the language works and what it means. It aims to be
complete and easy to understand for people who are not programmers. If you know
Java or Rust already, you may want to read
[Musi for Java developers](/docs/for_java_developers.md) or
[Musi for Rust developers](/docs/for_rust_developers.md).
